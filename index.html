Market Appraisal Plugin

This plugin provides the ability to book an appointment directly into RPS based on available timeslots. The appointment process can create a new contact if there isn't already one.

Valuer Diaries

The appointment diary allows for negotiators/valuers to be pre-configured and added to the list, which is presented to your client, for the market appraisal to be booked.

Sales

Defaults:
Valuer, Negotiator, Property Consultant, Senior Property Consultant, Sales Negotiator, Sales Associate

For this please use full names and ensure that you have at least one person from each of your offices.

Lettings

Defaults:
Valuer, Negotiator, Property Consultant, Senior Property Consultant, Lettings Negotiator, Lettings Associate

For this please use full names and ensure that you have at least one person from each of your offices.

Notifications

When an appoitnment is being booked, the following notification events are triggered:

On Screen Notification

Default: "To book a valuation, please enter your e-mail and postcode below".

If you would like this changed you will need to review with your web company.

Email Subject

Default:
“Your Valuation Request”

Default:
“Your Lettings Valuation Request”

Reply Address

Default:
no-reply@client-website.co.uk

We will provide technical details to your web company/mail provider to allow emails to be sent on behalf of your domain.

Email Content

* Default:
((Vendor))

Your valuation has been booked for ((hh:mm)) on ((DD MMMM)) at ((Property Full Address)).

If you have any queries or need to rearrange, please call me on ((Property Valuer)) or e-mail ((valuer email address)).

We've also created an account for you which you can log into here.

Kind regards
((Property Negotiator))

* double parentheses (()) denote where data will be used.

Reapit will by default use your existing email template layout (including headers & footers) and update the text accordingly.

The second paragraph is subject to the availability of the Vendor Tracker once fully configured.

Email Recipients:

It's possible to BCC (Blind Carbon Copy) the branch manager into these emails. Tick the box below if you'd like this.

SMS Text Message:

* Default:
SALES: “Your valuation with ((negotiator)) has been booked for ((date)) at ((location)). To rearrange this appointment, please call ((telephone)).”

LETTINGS: “Your lettings valuation with ((negotiator)) has been booked for ((date)) at ((location)). To rearrange this appointment, please call ((telephone)).”

* double parentheses (()) denote where data will be used.

Please keep in mind this is a SMS text message and as such has some limitations.

Global Settings

Appointment Lead Time:

This is the time before an appointment can be made. This is set to ensure an appointment cannot be made immediately.

Default:
6 hours

Appointment Buffer

This is used to set time before and after a pre-existing appointment to keep travel time accomodated.

Default:
30 minutes

Default Office

A default office is set so that if a postcode can't be infentified or is not within any office catchment area the message can be redirected to it.
